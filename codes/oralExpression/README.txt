Working Dependences:
- OpenCV 1.0
- Microsoft Visual Studio 6.0.
	(also working in OpenCV 1.1pre and Visual Studio 2008. 
	Some changes would be needed to make the software working in posterior versions of OpenCV)

Two versions:
1) From video file
2) Real-time from a WebCam device

Testing version 1):
	> build and execute the program using the filename as the first unique argument
	
Testing version 2):
	> build and execute the program with an installed Webcam
